Kerbal Statics

This mod provides the ability to add new launch sites and other buildings to your game.
NOTE: This mod REQUIRES the Making History DLC!

Features
- Setup a new launch site by simply planting a flag!
- Provides the option to set up a new launch site as either a launch pad or a runway.
- The new site can be established just for the current game, or for all games.
- Launch from the new site using the VAB/SPH launch site dropdown list.

References
Project Eagle by Blackbird Interactive: https://blackbirdinteractive.com/project-eagle/
Steam Page: https://store.steampowered.com/app/224000/Project_Eagle_A_3D_Interactive_Mars_Base/

---INSTALLATION---

Copy the contents of the mod's GameData directory into your GameData folder. Specifically you'll need:

GameData
	WildBlueIndustries
		KerbalStatics
	ModuleManager.dll (the latest version is included)

If these directories already exist, then delete the existing ones before installing the latest update.
